Cafe Data Project

Abstract

In today's world, living in a connected society is making the gathering of data easier and more accessible. It is now becoming possible to better detect trends and make predictions in people from this data and using simple machine learning, this detection can become automated. The results of this data and prediction models can be beneficial to a business owner to run their business.

Project Description

Using sample data from a cafeteria at Lexmark over a two month period, We are developing three seperate machine learning algorithms to predict outcomes of three different problems pertaining to a set of sample data from a cafeteria at Lexmark over a two month period. Using these algorithms, we will create a dashboard to allow the cafeteria owner manipulate the data in order to gather useful predictive information from the trends.
