cafe-predicting project

In today's day in age, the Internet of things is making gathering data on people easier. It is becoming possible to better detect trends and make predictions in people. Using simple machine learning, this detection can become automated. The results of this data and prediction models can be beneficial to a business owner to use this technology for running their business.

Using sample data from a cafeteria at Lexmark over a two month period, we are developing three seperate machine learning algorithms to predict outcomes of three seperate problems pertaining to the cafe data. Using these algorithms, we will create a dashboard to allow the cafeteria owner manipulate the data in order to gather useful predictive information from the trends.
